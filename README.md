# Quick Port Marketing Site

Single-page marketing experience for the Quick Port heavy cargo route intelligence platform.

## Highlights
- Modern hero with demo video CTA
- Mobile showcase cards (record, track, review, onboard, login)
- TailwindCSS + Next.js 12 foundation

## Updating Mobile Screens
1. Host each screenshot and grab the direct URL.
2. Edit `pages/index.js` and update the `mobileScreens` array with those URLs.
3. If the domain is new, add it to `images.domains` in `next.config.js`.

## Local Development
```bash
yarn
yarn dev
```

## Build
```bash
yarn build
```
